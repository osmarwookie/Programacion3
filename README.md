# EcoSystem

Proyecto desarrollado para la materia Programación 3.

## Arquitectura N-Capas

La solución está organizada en dos capas principales:

### EcoSystem.API
Capa de presentación encargada de exponer servicios HTTP mediante ASP.NET Core Web API.

### EcoSystem.Data
Capa de acceso a datos encargada de almacenar entidades, modelos y la comunicación con la base de datos.

## Estructura del proyecto

EcoSystem.sln

├── EcoSystem.API

└── EcoSystem.Data

## Tecnologías utilizadas

- .NET 8
- ASP.NET Core Web API
- C#
- Arquitectura N-Capas
